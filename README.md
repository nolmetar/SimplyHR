## How to use

*In the following statements, replace http://localhost:8080/ by your URL*

### Recruiter

1. Recruiters must have an account, can be created at:
> http://localhost:8080/signup

2. Recruiters can log in at:
> http://localhost:8080/login

3. When logged in recruiters are at:
> http://localhost:8080/navigateOfferR

4. Recruiters can see their offers at:
> http://localhost:8080/navigateOfferR

5. Recruiters must do their planning at:
> http://localhost:8080/doPlanning

6. Recruiters can create an offer at:
> http://localhost:8080/createOffer

7. Recruiters can see pending applications at:
> http://localhost:8080/viewCVs

8. Recruiters can see applications history at:
> http://localhost:8080/candidateHistory

9. Recruiters can modify their offer at :
> http://localhost:8080/createOffer

### Candidate

1. Candidates must input information at:

> http://localhost:8080/

> http://localhost:8080/candidateInfo

2. Candidates can see offers at:
> http://localhost:8080/navigateOfferC

3. Candidates can see a selected offer at:
> http://localhost:8080/selectOffer

4. When application is accepted by a recruiter, candidate can select a time slot at:
> http://localhost:8080/choicetimepassage
